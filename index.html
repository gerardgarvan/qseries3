Overview

A standalone single-binary REPL for q-series computation in number theory, partition theory, and modular forms. Reimplements core functionality of Frank Garvan's qseries Maple package in C++20 with zero external dependencies. All arithmetic is exact rational — no floating point anywhere in the math pipeline.

Exact Arithmetic

Arbitrary-precision integers and exact rationals. Every coefficient is computed precisely.

Andrews' Algorithm

Convert any q-series to its infinite product form via prodmake.

Relation Finding

Discover polynomial relations between q-series with findhom, findpoly, and more.

50+ Built-ins

Theta functions, eta products, Eisenstein series, Jacobi/triple/quintuple products, and number-theoretic utilities.

Based on Frank Garvan's qseries Maple package for research in partition theory and q-series.

Build

Requires a C++20 compiler. No package managers, no dependencies — just compile and run.

$ g++ -std=c++20 -O2 -static -o qseries src/main.cpp

Verify the build:

$ echo "partition(100)" | ./qseries
190569292

Linux

GCC 10+ or Clang 13+. Compile with the one-liner above. The -static flag produces a portable binary.

macOS

Apple Clang 14+ (Xcode 14+). Drop -static — Apple doesn't support fully static linking.

Windows

Use MSYS2/MinGW-w64 or Cygwin with g++. The binary runs natively from any terminal.
