Overview

A standalone single-binary REPL for q-series computation in number theory, partition theory, and modular forms. Reimplements core functionality of Frank Garvan's qseries Maple package in C++20 with zero external dependencies. All arithmetic is exact rational — no floating point anywhere in the math pipeline.

Exact Arithmetic

Arbitrary-precision integers and exact rationals. Every coefficient is computed precisely.

Andrews' Algorithm

Convert any q-series to its infinite product form via prodmake.

Relation Finding

Discover polynomial relations between q-series with findhom, findpoly, and more.

50+ Built-ins

Theta functions, eta products, Eisenstein series, Jacobi/triple/quintuple products, and number-theoretic utilities.

Based on Frank Garvan's qseries Maple package for research in partition theory and q-series.

Build

Requires a C++20 compiler. No package managers, no dependencies — just clone, compile, and run.

$ git clone https://github.com/gerardgarvan/qseries3.git && cd qseries3
$ g++ -std=c++20 -O2 -static -o qseries src/main.cpp

Verify the build:

$ echo "partition(100)" | ./qseries
190569292

Linux

GCC 10+ or Clang 13+. Compile with the one-liner above. The -static flag produces a portable binary.

macOS

Apple Clang 14+ (Xcode 14+). Drop -static — Apple doesn't support fully static linking.

Windows

Use MSYS2/MinGW-w64 or Cygwin with g++. The binary runs natively from any terminal.

Examples

Rogers–Ramanujan Identity

Compute the Rogers–Ramanujan continued fraction sum and recover its infinite product form using Andrews' algorithm. The denominators appear only at exponents ≡ ±1 (mod 5).

qseries> rr := sum(q^(n^2)/aqprod(q,q,n,50), n, 0, 8) 1 + q + q² + q³ + 2q⁴ + 2q⁵ + 3q⁶ + 3q⁷ + 4q⁸ + 5q⁹ + 6q¹⁰ + 7q¹¹ + 9q¹² + 10q¹³ + 12q¹⁴ + 14q¹⁵ + 17q¹⁶ + 19q¹⁷ + 23q¹⁸ + 26q¹⁹ + 31q²⁰ + 35q²¹ + 41q²² + 46q²³ + 54q²⁴ + 61q²⁵ + 70q²⁶ + 79q²⁷ + 91q²⁸ + 102q²⁹ + O(q⁵⁰) qseries> prodmake(rr, 50) 1 / (((1-q) (-q⁴+1) (-q⁶+1) (-q⁹+1) (-q¹¹+1) (-q¹⁴+1) (-q¹⁶+1) (-q¹⁹+1) (-q²¹+1) (-q²⁴+1) (-q²⁶+1) (-q²⁹+1) (-q³¹+1) (-q³⁴+1) (-q³⁶+1) (-q³⁹+1) (-q⁴¹+1) (-q⁴⁴+1) (-q⁴⁶+1) (-q⁴⁹+1)))

Partition Function

The partition generating function 1/(q;q)∞ encodes the number of partitions of n as the coefficient of qⁿ. The first 30 partition numbers, plus p(100) computed directly.

qseries> f := 1/etaq(1,30) 1 + q + 2q² + 3q³ + 5q⁴ + 7q⁵ + 11q⁶ + 15q⁷ + 22q⁸ + 30q⁹ + 42q¹⁰ + 56q¹¹ + 77q¹² + 101q¹³ + 135q¹⁴ + 176q¹⁵ + 231q¹⁶ + 297q¹⁷ + 385q¹⁸ + 490q¹⁹ + 627q²⁰ + 792q²¹ + 1002q²² + 1255q²³ + 1575q²⁴ + 1958q²⁵ + 2436q²⁶ + 3010q²⁷ + 3718q²⁸ + 4565q²⁹ + O(q³⁰) qseries> partition(100) 190569292

Theta → Eta Product

Identify a Jacobi theta function as a product of Dedekind eta functions using etamake.

qseries> etamake(theta3(50), 50) η(2τ)⁵ / (η(τ)⁻² η(4τ)⁻²)

Finding Relations: E₄² = E₈

Use findpoly to discover that the square of the Eisenstein series E₄ equals E₈ — a classical identity in the theory of modular forms.

qseries> x := eisenstein(2,50) 1 + 240q + 2160q² + 6720q³ + 17520q⁴ + 30240q⁵ + 60480q⁶ + 82560q⁷ + ... qseries> y := eisenstein(4,50) 1 + 480q + 61920q² + 1050240q³ + 7926240q⁴ + 37500480q⁵ + ... qseries> findpoly(x, y, 2, 3) -X₁²+X₂

Demo Script

Copy this shell script to run all the Garvan tutorial examples. Save as garvan-demo.sh alongside the qseries binary, then run with bash garvan-demo.sh. 

#!/usr/bin/env bash
# Garvan demo — runs qseries commands from qseriesdoc.
# Save alongside qseries binary, then: bash garvan-demo.sh
set -e
DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
BIN="$DIR/qseries.exe"
[ -f "$BIN" ] || BIN="$DIR/qseries"
[ -f "$BIN" ] || { echo "error: qseries not found"; exit 1; }

run() {
  for cmd in "$@"; do echo "qseries> $cmd"; done
  printf '%s\n' "$@" | "$BIN" 2>&1
}

# === Rogers-Ramanujan (qseriesdoc §3.1) ===
echo ""
echo "--- Rogers-Ramanujan ---"
run "rr := sum(q^(n^2)/aqprod(q,q,n,50), n, 0, 8)" \
    "series(rr, 50)" \
    "prodmake(rr, 40)"

# === Product conversion: qfactor (qseriesdoc §3.2) ===
echo ""
echo "--- qfactor T(8,8) ---"
run "set_trunc(64)" "t8 := T(8,8)" "qfactor(t8, 20)"

# === etamake on theta functions (qseriesdoc §3.3) ===
echo ""
echo "--- etamake theta functions ---"
run "set_trunc(100)" \
    "etamake(theta2(100), 100)" \
    "etamake(theta3(100), 100)" \
    "etamake(theta4(100), 100)"

# === jacprodmake Rogers-Ramanujan (qseriesdoc §3.4) ===
echo ""
echo "--- jacprodmake Rogers-Ramanujan ---"
run "rr := sum(q^(n^2)/aqprod(q,q,n,50), n, 0, 8)" \
    "jp := jacprodmake(rr, 40)" \
    "jac2prod(jp)"

# === findhom: Gauss AGM (qseriesdoc §4.1) ===
echo ""
echo "--- findhom: Gauss AGM ---"
run "set_trunc(100)" \
    "findhom([theta3(q,100), theta4(q,100), \
      theta3(subs_q(q,2),100), theta4(subs_q(q,2),100)], 2, 0)"

# === findnonhomcombo: Watson modular equation (§4.4) ===
echo ""
echo "--- Watson modular equation ---"
run "set_trunc(100)" \
    "xi := q^2*etaq(49,100)/etaq(1,100)" \
    "T := q*(etaq(7,100)/etaq(1,100))^4" \
    "findnonhomcombo(T^2, [T, xi], [1, 7], 0)"

# === Sifting: Rødseth identity (qseriesdoc §5) ===
echo ""
echo "--- Sifting: Rødseth ---"
run "set_trunc(200)" \
    "PD := etaq(2,200)/etaq(1,200)" \
    "PD1 := sift(PD,5,1,199)" \
    "etamake(PD1, 30)"

# === Triple product: Euler pentagonal (§6.1) ===
echo ""
echo "--- Triple product ---"
run "set_trunc(60)" "series(tripleprod(q,q^3,10), 60)"

# === Quintuple product and Euler dissection (§6.2) ===
echo ""
echo "--- Quintuple product ---"
run "set_trunc(500)" "EULER := etaq(1,500)" \
    "E0 := sift(EULER,5,0,499)" \
    "jp := jacprodmake(E0,50)" \
    "jac2prod(jp)"

# === Winquist identity (§6.3) ===
echo ""
echo "--- Winquist ---"
run "set_trunc(200)" "series(winquist(q^5,q^3,q^11,20), 60)"
