  <action>
Create src/series_omega.h. Include omega3.h and series.h (for Series type in mixed op).

Struct SeriesOmega:
- std::map&lt;int, Omega3&gt; c
- int trunc
- Omit q_shift for v1 (per instructions)

Factories (mirror Series):
- static SeriesOmega zero(int T)
- static SeriesOmega one(int T) — coeff 0 = Omega3(1,0)
- static SeriesOmega constant(Omega3 z, int T)
- static SeriesOmega q(int T) — coeff 1 = Omega3(1,0)

Helpers:
- Omega3 coeff(int n) const — returns c[n] or Omega3(0,0) if absent
- void setCoeff(int n, Omega3 v) — erase if zero, else store
- int minExp() const, int maxExp() const — mirror Series (for tests)
- void clean() — remove terms with e >= trunc or coeff isZero()

Arithmetic (propagate trunc = min(left.trunc, right.trunc)):
- SeriesOmega operator+(const SeriesOmega&amp; o) const — add coeffs per exponent, call clean
- SeriesOmega operator*(const SeriesOmega&amp; o) const — Cauchy product; only keep exp &lt; trunc (no terms beyond T-1)
- SeriesOmega operator*(const Omega3&amp; z) const — scalar multiply: for each (e,v), coeff[e] = z * v

Truncation:
- SeriesOmega truncTo(int T) const — copy coeffs with e &lt; min(trunc,T), set trunc = min(trunc,T)

Mixed op (free function in same header):
- SeriesOmega operator*(const Omega3&amp; z, const Series&amp; s) — for each (e, Frac v), result.coeff[e] = z * Omega3::fromRational(v). Use z * Omega3::fromRational(v) since Omega3::fromRational(v) = Omega3(v,0) and z * that gives correct Q(ω) scalar multiple. Set result.trunc = s.trunc.
</action>
