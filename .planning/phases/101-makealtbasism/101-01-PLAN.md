Add makeALTbasisM(k, T) after makebasisM (before makebasisPX).

Implementation (per CONTEXT.md locked decisions):
- Error: if (k &lt; 0 || k % 2 != 0) throw std::runtime_error("makeALTbasisM: k must be nonnegative even integer");
- s = k % 12; r = (k-s)/12 for s≠2; r = (k-14)/12 for s==2
- Es table: s=0→Series::one(T+5), s=2→EISENq(14,T+5), s=4→EISENq(4,T+5), s=6→EISENq(6,T+5), s=8→EISENq(8,T+5), s=10→EISENq(10,T+5)
- E6 = EISENq(6, T+5), D = DELTA12(T+5) (or call DELTA12 with T+5; if DELTA12 takes T and does internal T+30, pass T+5 for truncation — match makebasisM style)
- For i = 0 to r: f = (Es * E6.pow(2*r-2*i) * D.pow(i)).truncTo(T+5); f.trunc = T; f.clean(); FL.push_back(std::move(f));
- Return FL

Match DELTA12 usage: DELTA12 returns Series truncated internally. For intermediate precision use T+5 or T+10. Check DELTA12 signature — it takes T and returns series; call DELTA12(T+5) so internal eta computation has headroom. Then products truncate to T before push.

k=2: r=-1, loop for (int i=0; i&lt;=r; ++i) runs 0 times → return empty vector.
