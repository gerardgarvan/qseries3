Struct Omega3 { Frac a, b } represents a + b·ω with ω² = -ω-1 (ω = exp(2πi/3)).

Required API:
- Omega3() → (0,0)
- Omega3(Frac a, Frac b) — reduce/normalize after construction (a,b stored as-is; Frac is already reduced)
- static Omega3 omega() → Omega3(0,1)
- static Omega3 omega2() → Omega3(-1,-1) since ω² = -ω-1
- static Omega3 fromRational(Frac r) → Omega3(r, 0)

Arithmetic (use ω² = -ω-1):
- (a+bω) + (c+dω) = (a+c) + (b+d)ω
- (a+bω) - (c+dω) = (a-c) + (b-d)ω
- (a+bω)(c+dω) = ac + adω + bcω + bdω² = (ac - bd) + (ad+bc-bd)ω

Division: (a+bω)⁻¹ = ((a-b) - bω) / N where N = a² - ab + b². Throw std::invalid_argument on division by zero (isZero check). Normalize result via Frac::reduce (the inverse formula yields Frac coefficients that may need reduction).

Helper: conjugate() → Omega3(a-b, -b) (conjugate of a+bω in Q(ω) with ω̅ = ω² = -ω-1).
Helper: norm() → Frac (a² - ab + b²).
Helper: isZero() → a==0 && b==0.

Power: static Omega3 pow(Omega3 z, int k) — reduce k mod 3 for base omega/omega2; return omega^(k mod 3). For general z, use repeated multiplication (k≥0) or 1/pow(z,-k) (k&lt;0). For omega and omega2 bases, use table: ω^0=1, ω^1=ω, ω^2=ω²=(-1,-1).

str():
- (0,0) → "0"
- (1,0) → "1"
- (0,1) → "omega"
- (-1,-1) → "omega^2"
- General: "a + b*omega" if b&gt;0; "a - |b|*omega" if b&lt;0 (use b.abs() for coefficient, minus sign before term)
- Handle negative a: "-1 + omega", "-2 - 3*omega"
- Omit a when a=0 (except for "0"); omit "+ " when a=0 and b&gt;0
