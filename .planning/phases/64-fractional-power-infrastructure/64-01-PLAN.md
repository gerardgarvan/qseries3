## Tasks

### Task 1: BigInt::iroot and BigInt::bigpow helpers
**File:** `src/bigint.h`
**What:** Add two static helper methods to BigInt.

`BigInt::bigpow(BigInt base, int exp)` — raise BigInt to non-negative integer power via binary exponentiation. Needed by `rational_pow` to compute `a^p` before taking q-th root.

`BigInt::iroot(BigInt n, int k)` — compute the integer k-th root of `n` via binary search. Returns `r` such that `r^k == n`. Throws `std::invalid_argument` if `n` is not a perfect k-th power. Only handles non-negative `n` (caller handles sign). k must be >= 1.

**Binary search approach for iroot:**
- If n == 0 or n == 1, return n
- low = 1, high = n (tighten upper bound by clamping to avoid huge search space — use bit-length estimate)
- While low <= high: mid = (low+high)/2, p = bigpow(mid, k); compare p to n
- Return mid if p == n, else adjust bounds
- After loop, throw "not a perfect k-th power"

**Insert location:** After the `bigGcd` function (around line 407), before the `#endif`.

### Task 2: Frac::rational_pow helper
**File:** `src/frac.h`
**What:** Add `Frac rational_pow(Frac alpha) const` method.

Computes `(this)^alpha` where the result must be an exact rational. If the result is irrational (e.g., `3^(1/2)`), throws `std::invalid_argument`.

**Algorithm:**
1. If `alpha.den == BigInt(1)`: simple integer power
   - Extract `p` from `alpha.num`, compute `num^p / den^p`
   - If `p < 0`, swap num/den before powering
2. General case `alpha = p/q`:
   - Compute `num_p = BigInt::bigpow(this->num, |p|)`, `den_p = BigInt::bigpow(this->den, |p|)`
   - `num_root = BigInt::iroot(num_p, q_int)`, `den_root = BigInt::iroot(den_p, q_int)` — throws if not perfect
   - If `p < 0`: swap num_root/den_root
   - Return `Frac(num_root, den_root)`
3. Handle sign: if `this` is negative and `q` is even, throw (no real even root of negative)

**Insert location:** After the last operator method, before the `str()` method.

### Task 3: Series::powFrac core method
**File:** `src/series.h`
**What:** Add `Series powFrac(Frac alpha) const` method.

**Algorithm:**
1. **Integer shortcut:** If `alpha.den == BigInt(1)`, extract int and delegate to `pow(int)`.
2. **Empty series:** If `c.empty()`, return zero series (0^α = 0 for α > 0, error for α ≤ 0).
3. **Extract leading power:** `k = minExp()`. Build shifted series `f` with coefficients `f[e-k] = c[e]`, `f.trunc = trunc - k`.
4. **Factor out constant:** `c0 = f.coeff(0)`. If `c0.isZero()`, throw.
5. **Compute scale:** `scale = c0.rational_pow(alpha)` — throws if irrational.
6. **Normalize coefficients:** `g[j] = f.coeff(j) / c0` for j >= 1.
7. **Coefficient recurrence:**
   ```
   h[0] = Frac(1)
   for m = 1 to f.trunc - 1:
     sum = Frac(0)
     for j = 1 to m-1:
       coeff = ((alpha + Frac(1)) * Frac(j) - Frac(m))
       sum = sum + coeff * g[j] * h[m-j]
     h[m] = alpha * g[m] + sum / Frac(m)
   ```
   NOTE: Only the summation is divided by m, NOT the α·g[m] leading term.
8. **Scale and build result:**
   ```
   result.trunc = trunc - k
   for each h[m]: result.c[m] = scale * h[m]
   result.q_shift = (q_shift + Frac(k)) * alpha
   ```
9. **normalize_q_shift() and return.**

**Insert location:** After `pow(int n)` method (around line 280).

### Task 4: REPL dispatch for fractional exponents
**File:** `src/repl.h`
**What:** Update the `BinOp::Pow` case in `evalExpr` to fall back to `powFrac` when the exponent is not an integer.

**Current code (lines 1025–1030):**
```cpp
case BinOp::Pow: {
    int64_t expVal = evalToInt(e->right.get(), env, sumIndices);
    if (expVal > 10000 || expVal < -10000)
        throw std::runtime_error("pow: exponent magnitude too large (limit 10000)");
    return l.pow(static_cast<int>(expVal));
}
```

**New code:**
```cpp
case BinOp::Pow: {
    try {
        int64_t expVal = evalToInt(e->right.get(), env, sumIndices);
        if (expVal > 10000 || expVal < -10000)
            throw std::runtime_error("pow: exponent magnitude too large (limit 10000)");
        return l.pow(static_cast<int>(expVal));
    } catch (...) {
        EnvValue rv = eval(e->right.get(), env, sumIndices);
        Series rs = toSeries(rv, "pow exponent", env.T);
        if (rs.c.size() == 1 && rs.c.count(0) && rs.q_shift.isZero()) {
            return l.powFrac(rs.c.at(0));
        }
        throw std::runtime_error("pow: exponent must be an integer or rational constant");
    }
}
```

## Risks
- **BigInt::iroot binary search bounds:** Upper bound must be tight enough to avoid huge search. Use bit-length heuristic.
- **Frac arithmetic growth:** The recurrence involves fractions with growing denominators. GCD reduction after each step keeps sizes manageable.
- **Edge case:** `0^(1/2)` should return 0, not throw. Handle empty/zero series.
