## Tasks

<task id="1" title="Create rr_ids.h with RRG(1), RRH(1) sum implementations">

**Files:** `src/rr_ids.h`

**Action:**

1. Create `src/rr_ids.h` including `series.h`, `qfuncs.h`, `frac.h`, `convert.h`, `theta_ids.h`.
2. Implement `Series RRG1(int T)` = Σ_{n=0}^{floor(√T)} q^(n²)/(q;q)_n using `aqprod(q,q,n,T)` and `Series::pow(n*n)`. Sum until n*n < T.
3. Implement `Series RRH1(int T)` = Σ_{n=0} q^(n²+n)/(q;q)_n until n²+n < T.
4. Implement `Series RRG(int n, int T)`: for n==1 return RRG1(T); for n>1 return `geta(1, 5, n, T)` (JAC-based).
5. Implement `Series RRH(int n, int T)`: for n==1 return RRH1(T); for n>1 return `geta(2, 5, n, T)`.
6. Implement `Series geta(int g, int d, int n, int T)`:
   - GetaEXP = n * QP2(Frac(g,d)) * d / 2 (Frac arithmetic; result is rational)
   - Build JacFactor list: JAC(n*g, n*d, 1) / JAC(0, n*d, 1) (num = {(n*g, n*d, 1)}, den = {(0, n*d, 1)})
   - Series jac = jac2series(jac_list, T)
   - Apply q-power: result = q_var.powFrac(GetaEXP) * jac, truncated to T
   - Use `Series::constant(1,T)` and `q.pow(...)` for integer exponent; for fractional use powFrac on q^GetaEXP
7. Implement `Series RRGstar1(int T)` = Σ_{n=0} q^(n²)*(-q;q²)_n/(q²;q²)_n. Use `aqprod(q*(-1), q*q, n, T)` for (-q;q²)_n and `aqprod(q*q, q*q, n, T)` for (q²;q²)_n.
8. Implement `Series RRHstar1(int T)` = Σ_{n=0} q^(n*(n+2))*(-q;q²)_n/(q²;q²)_n.
9. Implement `Series RRGstar(int n, int T)` and `Series RRHstar(int n, int T)`: for n==1 return RRGstar1/RRHstar1; for n>1 use geta with modulus 8 (g,d pairs for GG from RESEARCH: G* uses (1,8),(4,8),(7,8), H* uses (3,8),(4,8),(5,8) — implement via JAC products or defer n>1 to later; n=1 suffices for findids type 1–2).

**Verify:** `make` succeeds; `echo "set_trunc(50); series(RRG(1,50),50)" | ./qseries` produces non-empty series; prodmake on RRG(1) yields denominators at ±1 mod 5.

**Done:** RRG1 and RRH1 return correct Rogers-Ramanujan series; geta produces correct JAC-based series for n>1.
</task>
