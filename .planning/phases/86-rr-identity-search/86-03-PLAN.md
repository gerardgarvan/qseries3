<task id="1" title="Implement GE, HE and ABCOND helpers in rr_ids.h">

**Files:** `src/rr_ids.h`

**Action:**

1. **GE(a) and HE(a)** for Rogers-Ramanujan modulus 5:
   - GE(a) = (5*a*a - 5*a + 2) / 10  — closed form from Ramanujan-Robins-Sellers
   - HE(a) = (5*a*a + 5*a + 2) / 10
   - Return Frac for exact arithmetic; use Frac(5*a*a - 5*a + 2, 10) etc.
   - Ensure denominator divides evenly (5a²±5a+2 is even for integer a).

2. **abcond_type1(p, a)**: ABCOND = GE(p)+HE(a)-(GE(a)+HE(p)). Return true iff result is integer (den==1 and no fractional part).

3. **abcond_type2(a, p)**: ABCOND = GE(a)+GE(p)-(HE(a)+HE(p)). Return true iff integer.

4. Helper to check Frac is integer: `fracIsInteger(Frac x)` → x.den == 1.

**Verify:** abcond_type1(2,1) and abcond_type1(5,2) etc. return true for valid Slater identities; abcond_type2(1,1), abcond_type2(1,2) for type 2.

**Done:** GE, HE, abcond_type1, abcond_type2 implemented and correct.
</task>

<task id="2" title="Implement findids_type1 and findids_type2 in rr_ids.h">

**Files:** `src/rr_ids.h`

**Action:**

1. **findids_type1(T, acc=10)**:
   - Result: `std::vector<std::tuple<int,int,int>>` for (p, a, c1)
   - Loop: n from 2 to T
   - For each divisor p of n: a = n/p
   - Skip if a==p or a>p or gcd(a,p)!=1
   - For c1 in {-1, 1}:
     - If !abcond_type1(p, a) continue
     - SYMF = RRG(p,T) * RRH(a,T) + Frac(c1) * RRG(a,T) * RRH(p,T)
     - If checkid(SYMF, T, acc) returns ok, push {p, a, c1}
   - Return list

2. **findids_type2(T, acc=10)**:
   - Result: `std::vector<std::tuple<int,int,int>>` for (a, p, c1)
   - Loop: n from 1 to T
   - For each divisor p of n: a = n/p
   - Skip if a>p or gcd(a,p)!=1
   - For c1 in {-1, 1}:
     - If !abcond_type2(a, p) continue
     - SYMF = RRG(a,T) * RRG(p,T) + Frac(c1) * RRH(a,T) * RRH(p,T)
     - If checkid(SYMF, T, acc) returns ok, push {a, p, c1}
   - Return list

3. **findids(type, T)** dispatcher: type 1 → findids_type1, type 2 → findids_type2; type 3–10 return empty (deferred).

**Verify:** findids_type1(20) returns ≥3 entries; findids_type2(20) returns ≥2 entries. Known: (2,1,-1), (5,2,1), (5,3,-1) etc. for type 1; (1,1,1), (1,2,-1) for type 2.

**Done:** findids(1,20) and findids(2,20) discover expected identities.
</task>
