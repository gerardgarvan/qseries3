## Tasks

<task id="1" title="Implement checkid in rr_ids.h">

**Files:** `src/rr_ids.h`

**Action:**

1. Implement `checkid(f, T, acc = 10)`:
   - Input: Series f, int T, int acc (default 10)
   - ldq = f.minExp(); if ldq < 0 return empty (optional: allow negative for shifted series)
   - g = f.truncTo(T)
   - c0 = g.coeff(ldq); if c0.isZero() return empty
   - h = g / (Series::constant(c0, T) * Series::q(T).pow(ldq)); h = h.truncTo(T) — normalize to constant 1
   - If h.coeff(0) != 1 or h.minExp() > 0, return empty
   - a_map = prodmake(h, T); if a_map.empty() return empty
   - plx = max(|a_n|) over a_map; if any a_n has non-integer denominator, return empty
   - If plx >= acc, return empty
   - Run etamake on g (the un-normalized series after q^ldq removal) — etamake expects constant term; use g = f.truncTo(T); g_normalized = g / (c0 * q^ldq); then etamake(g_normalized, T) gives exponents; reattach q^ldq in output
   - Return: optional or struct holding (eta_list, ldq) — format as "q^ldq * eta_product"

2. Define result type: `struct CheckidResult { bool ok; int ldq; std::vector<std::pair<int,Frac>> eta; }` or use std::optional<std::vector<std::pair<int,Frac>>> with ldq passed separately. For REPL display: if ok, format as "q^N * η(k1)^e1 η(k2)^e2 ...".

3. Use prodmake (not newprodmake) because we normalize to constant 1 explicitly. prodmake requires constant term 1.

**Verify:** Unit logic: for f = RRG(1)*RRH(1) with T=50, checkid returns ok and non-empty eta list.

**Done:** checkid correctly identifies eta-quotients and rejects non-eta series.
</task>
