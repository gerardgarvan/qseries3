  <action>
Implement tripleprod_symbolic(const Series& q, int T) -> BivariateSeries in qfuncs.h.

Formula: Σ (-1)^n z^n q^(n(n-1)/2) for n such that n(n-1)/2 < T.

Algorithm:
1. Iterate n = 0, 1, 2, ... while n*(n-1)/2 < T
2. q_exp = n*(n-1)/2
3. z_exp = n
4. coeff = (-1)^n
5. b.c[{z_exp, q_exp}] += Frac(coeff)  (or set; only one term per (z_exp,q_exp))
6. Set b.trunc = T

Handle n=0: 0*(0-1)/2 = 0, z^0 q^0, coeff 1. So constant term 1.
</action>

  <done>tripleprod_symbolic returns correct BivariateSeries for T=10 (constant 1 + terms up to q^36 for n=9)</done>

<task type="auto">
  <name>Task 3: quinprod_symbolic implementation</name>
  <files>src/qfuncs.h</files>
  <action>
Implement quinprod_symbolic(const Series& q, int T) -> BivariateSeries in qfuncs.h.

Formula: Σ ((-z)^(-3n) - (-z)^(3n+1)) q^(n(3n+1)/2) for n in Z (both positive and negative), truncated q_exp < T.

For n >= 0: q_exp = n(3n+1)/2 (pentagonal numbers: 0, 2, 7, 15, ...)
  - Term 1: (-z)^(-3n) → coefficient (-1)^(-3n) = (-1)^n, z_exp = -3n
  - Term 2: -(-z)^(3n+1) → coefficient -(-1)^(3n+1) = (-1)^n, z_exp = 3n+1
  So: (-1)^n * (z^(-3n) - z^(3n+1)) for n >= 0

For n < 0: let m = -n, m > 0. Then -3n = 3m, 3n+1 = 1-3m.
  q_exp = n(3n+1)/2 = -m(1-3m)/2 = m(3m-1)/2 (pentagonal: 1, 5, 12, 22, ...)
  Terms: (-z)^(3m) - (-z)^(1-3m) → similar structure

Unify: iterate n over integers where q_exp = n(3n+1)/2 < T. For each n:
  - q_exp = n*(3*n+1)/2 (valid for negative n: n=-1 → 1, n=-2 → 5, etc.)
  - Add (-1)^n * z^(-3n) with coeff (-1)^n
  - Add -(-1)^n * z^(3n+1) with coeff -(-1)^n
  Actually: ((-z)^(-3n) - (-z)^(3n+1)) = (-1)^n * z^(-3n) - (-1)^(3n+1) * z^(3n+1)
  = (-1)^n z^(-3n) + (-1)^n z^(3n+1) since (-1)^(3n+1) = -(-1)^n.
  Wait: -(-z)^(3n+1) = -(-1)^(3n+1) z^(3n+1) = (-1)^(3n+2) z^(3n+1) = (-1)^n z^(3n+1) for integer n.
  So both terms have coeff (-1)^n. Term 1: z^(-3n), Term 2: z^(3n+1).

For n=0: z^0 - z^1, coeff 1 each → add 1 at (0,0) and -1 at (1,0).
For n=1: q_exp=2, (-1)*(z^(-3) - z^4) → -1 at (-3,2), +1 at (4,2).
For n=-1: q_exp=1, (-1)^(-1)*(z^3 - z^(-2)) = -z^3 + z^(-2) → -1 at (3,1), +1 at (-2,1).

Iterate n from -N to N such that |n(3n+1)/2| < T. n(3n+1)/2 is 0,1,2,5,7,12,15,... for n=0,1,-1,2,-2,3,-3,...
</action>
  <verify>make 2>&1 | tail -3</verify>
  <done>quinprod_symbolic returns correct BivariateSeries for T=3 (terms at q^0, q^1, q^2)</done>
