  **tutorial/rogers-ramanujan.md** — title "Rogers-Ramanujan Identities", description "Computing and verifying the Rogers-Ramanujan continued fraction identities"

  Source: qseriesdoc.md §3.1 (prodmake) and §3.4 (jacprodmake).

  Content sections:

  1. **The Rogers-Ramanujan Identity** — State the identity with KaTeX:
     $$\sum_{n=0}^{\infty} \frac{q^{n^2}}{(q;q)_n} = \prod_{n=1}^{\infty} \frac{1}{(1-q^{5n-1})(1-q^{5n-4})}$$
     Brief explanation: the left side generates partitions whose parts differ by at least 2; the right side generates partitions into parts ≡ ±1 (mod 5). The identity says these counts are equal for every n.

  2. **Computing the series** — REPL example:
     ```
     qseries> rr := sum(q^(n^2)/aqprod(q,q,n,50), n, 0, 8)
     ```
     Show the output (first ~10 terms). Explain that we sum to n=8 — higher n contribute only to very high-order terms due to the $q^{n^2}$ factor.

  3. **Identifying the product with prodmake** — REPL example:
     ```
     qseries> prodmake(rr, 40)
     ```
     Show the output: product with denominators only at exponents ≡ ±1 (mod 5). Explain: "We have computationally rediscovered the right side of the Rogers-Ramanujan identity!"

  4. **Jacobi product form with jacprodmake** — REPL example:
     ```
     qseries> jp := jacprodmake(rr, 40)
     ```
     Show the output: `JAC(0,5,inf) / JAC(1,5,inf)`. Explain the JAC notation.
     ```
     qseries> jac2prod(jp)
     ```
     Show the output in $(q^a;q^b)_\infty$ notation.

  5. **The second Rogers-Ramanujan identity** — State:
     $$\sum_{n=0}^{\infty} \frac{q^{n(n+1)}}{(q;q)_n} = \prod_{n=1}^{\infty} \frac{1}{(1-q^{5n-2})(1-q^{5n-3})}$$
     Show the verification:
     ```
     qseries> rr2 := sum(q^(n*(n+1))/aqprod(q,q,n,50), n, 0, 8)
     qseries> prodmake(rr2, 40)
     ```
     Note that now denominators appear at ≡ ±2 (mod 5).

  6. **Andrews' prodmake algorithm** — Brief overview of the algorithm (from qseriesdoc.md §3):
     Given $1 + \sum_{n=1}^{\infty} b_n q^n = \prod_{n=1}^{\infty} (1-q^n)^{-a_n}$, take logarithmic derivatives to get a recurrence for the $a_n$.
     State the recurrence (KaTeX) but don't reproduce the full proof — reference Garvan's tutorial and Andrews' book [q-series, §10.7].

  **tutorial/theta-functions.md** — title "Theta & Eta Functions", description "Theta functions as eta products and the etamake algorithm"

  Source: qseriesdoc.md §2.2 (theta/eta definitions) and §3.3 (etamake).

  Content sections:

  1. **Jacobi theta functions** — Define $\theta_2$, $\theta_3$, $\theta_4$ with KaTeX:
     $$\theta_2(q) = \sum_{n=-\infty}^{\infty} q^{(n+1/2)^2}, \quad \theta_3(q) = \sum_{n=-\infty}^{\infty} q^{n^2}, \quad \theta_4(q) = \sum_{n=-\infty}^{\infty} (-1)^n q^{n^2}$$
     Briefly explain their importance in modular form theory.

  2. **The Dedekind eta function** — Define:
     $$\eta(\tau) = q^{1/24} \prod_{n=1}^{\infty} (1-q^n), \quad q = e^{2\pi i \tau}$$
     Explain that `etaq(k, T)` computes $\prod_{n=1}^{\infty}(1-q^{kn})$ (the eta product without the $q^{1/24}$ factor).
     REPL example:
     ```
     qseries> etaq(1, 20)
     ```
     Show Euler's pentagonal number theorem coefficients.

  3. **Theta functions as eta products** — The key result. Use `etamake` to identify:
     ```
     qseries> etamake(theta3(100), 100)
     ```
     Show output: $\theta_3 = \eta(2\tau)^5 / (\eta(\tau)^2 \eta(4\tau)^2)$.
     ```
     qseries> etamake(theta4(100), 100)
     ```
     Show output: $\theta_4 = \eta(\tau)^2 / \eta(2\tau)$.

     State the well-known identities in KaTeX:
     $$\theta_3(q) = \frac{\eta(2\tau)^5}{\eta(\tau)^2 \eta(4\tau)^2}, \quad \theta_4(q) = \frac{\eta(\tau)^2}{\eta(2\tau)}$$

  4. **The etamake algorithm** — Brief explanation: given a q-series $1 + c q^k + \cdots$, recursively multiply by $\text{etaq}(k)^{-c}$ until all terms agree. This greedy approach works because eta products have a specific structure.

  5. **Partition generating functions as eta products** — Show the partition function:
     $$\sum_{n=0}^{\infty} p(n) q^n = \frac{1}{\eta(\tau)} \cdot q^{-1/24} = \prod_{n=1}^{\infty} \frac{1}{1-q^n}$$
     REPL:
     ```
     qseries> p := 1/etaq(1, 50)
     qseries> series(p, 15)
     ```
     Show first ~15 partition numbers.

  6. **p-core partition generating functions** — Mention:
     $$\sum_{n=0}^{\infty} a_p(n) q^n = \prod_{n=1}^{\infty} \frac{(1-q^{pn})^p}{1-q^n}$$
     Brief example with p=5.

  **tutorial/partition-identities.md** — title "Partition Identities", description "Sifting coefficients and discovering partition congruences"

  Source: qseriesdoc.md §3.2 (qfactor/T(r,n)) and §5 (sifting).

  Content sections:

  1. **The sift operation** — Explain what sifting does:
     Given $A(q) = \sum a_n q^n$, `sift(A, m, r)` extracts $\sum a_{mn+r} q^n$.
     This is the key tool for studying partition congruences like Ramanujan's:
     $$p(5n+4) \equiv 0 \pmod{5}$$

  2. **Distinct partitions mod 5** — Worked example from qseriesdoc.md §5. Let $pd(n)$ = number of partitions of n into distinct parts:
     $$\sum_{n=0}^{\infty} pd(n) q^n = \prod_{n=1}^{\infty} (1+q^n) = \frac{\prod(1-q^{2n})}{\prod(1-q^n)}$$
     REPL:
     ```
     qseries> PD := etaq(2, 200) / etaq(1, 200)
     qseries> PD1 := sift(PD, 5, 1, 38)
     qseries> etamake(PD1, 38)
     ```
     Show that $pd(5n+1)$ has a nice eta product form:
     $$\sum pd(5n+1) q^n = \frac{\eta(5\tau)^3 \eta(2\tau)^2}{\eta(10\tau) \eta(\tau)^4}$$
     This is due to Rødseth.

  3. **Ramanujan's most beautiful identity** — State:
     $$\sum_{n=0}^{\infty} p(5n+4) q^n = 5 \prod_{n=1}^{\infty} \frac{(1-q^{5n})^6}{(1-q^n)^5}$$
     Hardy and MacMahon called this Ramanujan's most beautiful identity. Show verification with sift:
     ```
     qseries> p := 1/etaq(1, 200)
     qseries> p5 := sift(p, 5, 4, 38)
     qseries> etamake(p5, 38)
     ```

  4. **The T(r,n) functions and qfactor** — From qseriesdoc.md §3.2. Andrews defined rational functions T(r,n) used in Rogers' original proof:
     ```
     qseries> set_trunc(64)
     qseries> t8 := T(8, 8)
     qseries> qfactor(t8, 20)
     ```
     Show that T(8,8) factors as a q-product. State the result:
     $$T(8,8) = \frac{(q^9;q)_3 (1-q^{16}) q^6}{(q;q)_4}$$

  5. **Euler's product and the quintuple product** — From qseriesdoc.md §6 (briefly). Dissect Euler's product mod 5:
     $$\prod (1-q^n) = E_0(q^5) + q E_1(q^5) + q^2 E_2(q^5)$$
     (Note $E_3 = E_4 = 0$ by the pentagonal number theorem.)
     Show using sift and jacprodmake to identify $E_0$:
     ```
     qseries> EULER := etaq(1, 500)
     qseries> E0 := sift(EULER, 5, 0, 98)
     qseries> jacprodmake(E0, 50)
     ```
     Relate to the quintuple product identity.

  **tutorial/modular-equations.md** — title "Modular Equations", description "Discovering polynomial relations among theta functions and modular forms"

  Source: qseriesdoc.md §4 (findhom, findhomcombo, findnonhomcombo, findpoly).

  Content sections:

  1. **The idea** — If q-series are modular forms, they satisfy polynomial relations. qseries converts this to linear algebra: expand all monomials of degree n as q-series, form a matrix, compute the kernel. Nontrivial kernel vectors correspond to relations.

  2. **Gauss AGM and theta relations (findhom)** — From qseriesdoc.md §4.1. Consider $\theta_3(q), \theta_4(q), \theta_3(q^2), \theta_4(q^2)$:
     ```
     qseries> findhom([theta3(100), theta4(100), theta3(subs_q(q,2),100), theta4(subs_q(q,2),100)], 2, 0)
     ```
     Show the output: two quadratic relations. State them in KaTeX:
     $$\theta_3(q^2)^2 = \frac{\theta_3(q)^2 + \theta_4(q)^2}{2}, \quad \theta_4(q^2)^2 = \theta_3(q) \theta_4(q)$$
     These are Gauss's parametrization of the arithmetic-geometric mean.

  3. **Eisenstein series and eta products (findhomcombo)** — From qseriesdoc.md §4.2. Define Eisenstein series $U_{p,k}$ and eta product basis functions $B_1, B_2$. Show that $U_{5,6}$ is a homogeneous cubic in $B_1, B_2$:
     ```
     qseries> B1 := etaq(1,50)^5/etaq(5,50)
     qseries> B2 := q*etaq(5,50)^5/etaq(1,50)
     ```
     Explain that computing $U_{5,6}$ requires the `sum` and `legendre` functions:
     ```
     qseries> f := sum(sum(legendre(m,5)*n^5*q^(m*n), n, 1, 50), m, 1, 50)
     qseries> findhomcombo(f, [B1, B2], 3, 0)
     ```
     Show the result: $U_{5,6} = B_1^2 B_2 + 40 B_1 B_2^2 + 335 B_2^3$.
     Note: If nested sum is not supported, explain the workaround (pre-compute the Eisenstein series coefficient-by-coefficient using a loop or helper).

  4. **Watson's modular equation (findnonhomcombo)** — From qseriesdoc.md §4.4. Define:
     $$\xi = \frac{\eta(49\tau)}{\eta(\tau)}, \quad T = \left(\frac{\eta(7\tau)}{\eta(\tau)}\right)^4$$
     Express $T^2$ as a polynomial in $T$ and $\xi$:
     ```
     qseries> xi := q^2*etaq(49,100)/etaq(1,100)
     qseries> TT := q*(etaq(7,100)/etaq(1,100))^4
     qseries> findnonhomcombo(TT^2, [TT, xi], [1, 7], 0)
     ```
     Show the output: Watson's modular equation used to prove Ramanujan's partition congruences for powers of 7.

  5. **Finding polynomial relations (findpoly)** — From qseriesdoc.md §4.5. Brief example showing `findpoly(x, y, 3, 1)` for a polynomial relation between two modular functions. Show the output: $(X+6)^3 Y - 27(X+2)^2 = 0$.

  For all examples: translate Maple syntax to our REPL syntax. Use `:=` for assignment, our function names, our output format. Do NOT include Maple-specific constructs like `with(qseries):`, semicolons, or `RETURN()`.
